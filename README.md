# My_first_netflix_clone

# 🎬 Netflix India Clone (HTML & CSS)

A fully responsive **Netflix India landing page clone** built using pure **HTML5 and CSS3**.

This project replicates the official Netflix homepage UI for learning and practicing frontend development skills.

---

## 🚀 Project Preview

This clone includes:

- Hero section with background overlay
- Navigation bar with language & sign-in button
- TV + video preview section
- Mobile download section
- Watch everywhere section
- Kids profile section
- FAQ accordion-style layout (UI only)
- Footer with multiple columns
- Clean Netflix-inspired design

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Layout Techniques
